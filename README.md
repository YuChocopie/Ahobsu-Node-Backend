# MashUp Node JS Project

## 매일매일 미션을 수행하며 카드를 수집해 보자

### 주요 기능

- 로그인 회원가입
  - 구글 로그인
  - 애플 로그인
- 매일 새로운 미션 제공
- 주별로 새로운 카드 세트 제공
- 미션 재발급 기능(제한적으로 제공)
